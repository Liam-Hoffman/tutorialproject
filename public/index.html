React introductory

React is a javascript library that developers can use to make user interfaces, these interfaces are usually used on single page applications, much like the website you are seeing now. The user interface application of react is that of being able to create reusable components that can be reused throughout the project. These components also allow the webpage to change the data on the page without reloading the page. This is achievable because react is fast, simple, and easily scalable. React takes advantage of JSX files, which is just javascript that allows HTML. You may have seen react before but never noticed, facebook is build using react.

Installing React

Navigate to https://www.npmjs.com/get-npm

Download and Install NPM

If you do not already have it installed, install visual studio code

Once NPM and VC goto Terminal>New Terminal and type "npm create-react-app my-app" and hit enter

Now type NPM start and your base react app will open!
